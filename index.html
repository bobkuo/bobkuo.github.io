About Me

我曾任職於研華「優店聯網」團隊，負責打造物聯網數據資料庫，服務客戶包括 IKEA、家樂福、新東陽等品牌。多年後，我決定轉職前端工程師，參加半年就業培訓課程，系統學習 Vue 3、Quasar、JavaScript、HTML5、CSS3、Git、REST API 並完成多個實作專案。

培訓期間，我開發了電商網站、插畫作品展示平台與地圖應用 LINE Bot，掌握 SPA 架構、Vue Router、Pinia 狀態管理、RWD 響應式設計與動畫特效，能獨立完成網站開發與 API 串接。

在天才視野公司時，我曾自行組裝數據收集設備進行實測，協助團隊提前發現產品問題，降低開發成本；這也展現我遇到挑戰能積極解決的能力。

我樂於合作與分享，曾擔任尾牙主持人及福委會主委，籌辦旅遊與單車活動凝聚團隊向心力。憑藉後端背景與前端專業，我希望為貴公司打造更流暢、美觀的使用者體驗。
